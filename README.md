# PipDeck

PipDeck is a lightweight graphical Python package manager built with Tkinter.

It provides a clean desktop interface for managing Python packages across multiple Python installations without needing to manually use terminal pip commands.

---

# Features

- Detects multiple Python installations automatically
- Displays installed packages with versions
- Package search and filtering
- Install packages using GUI
- Upgrade installed packages
- Remove installed packages
- Fetch package descriptions directly from PyPI
- Console output viewer
- Automatic pip detection and installation
- Cross-platform architecture
- Lightweight single-file application

---

# Supported Platforms

- Linux
- Windows
- Other Python-supported desktop platforms
